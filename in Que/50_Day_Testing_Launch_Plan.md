## 🧠 Under the Hood (The Tech Specs)
For the curious, these agents aren't just "magic boxes." They are built with a focus on simplicity and local hosting:
* **The Brain**: The coaching agents share a knowledge base pulled from a public GitHub repo (`agentic-brain`), processed using **Voyage AI `voyage-3-lite`** embeddings [cite: 1.0].
* **The Framework**: Built using **Mastra** for agent orchestration and **Next.js** for the frontend [cite: 1.0].
* **The Models**: I’m running **Claude 4.5 Haiku** for the coaching agents to keep responses fast and efficient, while the Librarian uses **Claude 4.6 Opus** for deeper biographical context [cite: 1.0].
* **Smart Routing**: Instead of an expensive LLM constantly guessing your intent, a **Regex keyword classifier** directs your message to the right specialist [cite: 1.0].
* **Hosting**: The system runs on a **Hostinger VPS** utilizing a **Docker manager** [cite: User Summary].
