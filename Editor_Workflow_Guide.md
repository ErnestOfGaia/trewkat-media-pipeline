# Editor Workflow: How to Edit and Sync Documents

This guide explains how to pull the launch plan to your preferred platform (GitHub Desktop, Google Docs, or VS Code with CoPilot) and push your changes back to the repository.

---

## Option A: Professional Editor Workflow (Recommended)
*Best if using VS Code + GitHub CoPilot.*

1. **Pull (Download):**
   - Open **GitHub Desktop** and select the `trewkat-media-pipeline` repository.
   - Click **Fetch origin** and then **Pull origin** to get the latest `50_Day_Testing_Launch_Plan.md`.
2. **Edit:**
   - Open the file in **VS Code**.
   - Use **CoPilot** (Cmd+I or Ctrl+I) to highlight text and ask: *"Rewrite this to sound more like a local tradesperson"* or *"Check for grammar and markdown errors."*
3. **Push (Upload):**
   - Go back to **GitHub Desktop**.
   - You will see your changes listed. Write a brief summary (e.g., "Edited for brand voice and clarity").
   - Click **Commit to main**, then click **Push origin**.

---

## Option B: Simplified Workflow (Google Docs)
*Best for pure text editing and collaboration.*

1. **Pull:**
   - Go to the file on [GitHub](https://github.com/ErnestOfGaia/trewkat-media-pipeline/blob/main/50_Day_Testing_Launch_Plan.md).
   - Click the **Raw** button. Copy the entire text.
   - Paste the text into a new **Google Doc**.
2. **Edit:**
   - Turn on **"Suggesting"** mode.
   - Perform your edits. If you need AI help, use the "Help me write" feature in Google Docs to simplify complex sentences.
3. **Push:**
   - Copy the final edited text from Google Docs.
   - Go back to the GitHub file link.
   - Click the **Pencil icon** (Edit this file).
   - Paste your new text over the old text.
   - Scroll down, add your commit message, and click **Commit changes**.

---

## Option C: Direct GitHub Editing
*Best for quick fixes.*

1. Go to the [50_Day_Testing_Launch_Plan.md](https://github.com/ErnestOfGaia/trewkat-media-pipeline/blob/main/50_Day_Testing_Launch_Plan.md) link.
2. Click the **Pencil icon** to edit directly in the browser.
3. Make your changes and click **Commit changes** at the bottom.
